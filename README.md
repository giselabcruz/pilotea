# pilotea
AI-powered guide for life administration and personal procedures.

This is a work in progress to help me with my life admin and personal procedures using AI Agents.

Nowadays, people study since 3 years old until 22, 23...30 years old, and then they start working. A lot of studies, but no one teaches you how to deal with all the bureaucracy and admin stuff in life. 

This is why I created pilotea. 

# How it works
Introduce your problem, the AI agent will ask you some questions to understand your problem better, and then it will provide you with a step-by-step guide to solve it. 

The Agent knows what is the problem of that you have, and it will send you to the agent that can help you with that problem. 

For example, if you have a problem with your taxes, it will send you to the tax agent. If you have a problem with your payslip, it will send you to the Finance agent. 

If you have some problem that involves more areas, the agents will communicate with each other to solve your problem.
